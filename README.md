# admin-app
Simple admin dashboard app created with Node.js, Express.js, ReactJS, MySQL and TypeScript.

## Backend
* Node.js
* Express
* TypeOrm

## Database
* MySQL

## Installation
1. Server setup.

 In backend folder change `typeorm.example.json` to `typeorm.json` and fill  your database connection.
 ```
 $ cd backend
 $ npm install
 $ npm start
 ```
